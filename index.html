Camera Color Picker

Camera Color Picker is a free and open source Android app that lets you capture, in real time, the colors around you using the camera of your device.

Road map

  • Add the possiblity to choose the default color code format. (html/rgb/hsv).
  • Add a button to turn on/off the flash of the camera.
  • Add a confirmation dialog before deleting a color.
  • Add a subtle message when a color is successfully saved.
  • Add french translation.
  • Add the possibility to create palettes with current saved colors.
  • Add a swipe to remove mechanism for deleting colors from the list.
  • Support tablets

Version 0.0.1 (2015-02-21)

  • Capture and save the colors with your device camera.
  • List all the colors saved.
  • Copy the html code of a color to the clipboard.
